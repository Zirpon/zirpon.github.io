算法时间复杂度

常数阶 O(1)

线性阶 O(n)

对数阶 O(logn)

${2^x} = n => x = log_2n$

平方阶 $O({n^2})$

$n+(n-1)+(n-2)+{\cdots}+1 =\frac{n(n+1)}{2}=\frac{n^2}{2}+\frac{n}{2}$

$O(n^2)$

常见时间复杂度

$O(1) < O(logn) < O(n) < O(nlogn) < O(n^2) < O(n^3) < O(2^n)< O(n!) < O(n^n)$
